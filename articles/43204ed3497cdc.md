まずは任意の場所にディレクトリを作る
(今回はgo-minimum-siteというディレクトリを作成)
```
$ mkdir go-minimum-site
```

作成したディレクトリに移動し、go.modを作成する
```
$ go mod init go-minimum-site
```

今回に必要なファイルを作成する
```
$ touch Dockerfile main.go index.html user.html
```
