```
import (
	"fmt"
	"time"
)

func main() {
	currentTime := time.Now()

	fmt.Println("Today is ", currentTime.Format("2006-01-02"))
}
```

```
Today is  2009-11-10
```

2006-01-01と2006-01-03は同じ値になる
2005-12-31は10000-1110-1111となる
